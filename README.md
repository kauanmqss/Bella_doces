# Bella Doces
Site de cadastro de produtos de uma loja de doces

Como usar:

1 - Página de Login:

◦ Clique em “Cadastrar” para criar um novo usuário.
◦ O e-mail deve estar em um formato válido (ex: exemplo@gmail.com).
◦ A senha deve conter no mínimo 8 caracteres, incluindo pelo menos um número.

2 - Acesso ao Sistema:

◦ Após o cadastro, é possível realizar login como usuário padrão para visualizar os produtos cadastrados.
◦ Também é possível acessar como administrador, utilizando as credenciais:
◦ Usuário: admin
◦ Senha: admin123

3 - Página do Administrador:

◦ Permite cadastrar novos produtos, informando:
◦ Nome do produto
◦ Preço
◦ URL da imagem

4 - Página de Relatórios:

◦ Exibe todos os produtos cadastrados.
◦ Possibilita editar ou excluir itens conforme necessário.
◦ Página de Vendas:
◦ Mostra a lista de produtos disponíveis.
◦ Permite alterar o status de cada produto entre “Disponível” e “Esgotado” — essa informação é refletida automaticamente na visualização do usuário padrão.
